Code snippet: python3 -m venv ~/myherozero
python3 - only entering python3 in your terminal opens the python environment, where you can run piece of python code. 
-m - this command allows python to find a module or a python file and run it from the command line.
venv -  Creates a virtual environment. A virtual environemnt allows you to interact with the computer. In this sequence, it creates the target the directory
and places a couple files inside of it. 
~/myherozero -  This is the name of the created virtual environment. The ~/ indicates the path, starting from the home directory.
